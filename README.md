## Project Description
This project is a responsive personal portfolio website created during Week 2 of the IYF Weekend Academy. It demonstrates HTML and CSS skills, including the CSS Box Model, Flexbox, CSS Grid, responsive design, and basic styling.

## Technologies Used
- HTML5
- CSS3
- GitHub
- GitHub Pages

## Features
- Responsive navigation bar
- About, Projects, and Contact pages
- CSS Box Model practice page
- Flexbox card layout
- CSS Grid gallery
- Hover effects and responsive design

## How to Run
1. Clone this repository.
2. Open `index.html` in your web browser.

## Lessons Learned
- How to style web pages with CSS.
- How to use the CSS Box Model.
- How to create layouts with Flexbox and CSS Grid.
- How to make web pages responsive.
- How to publish a website using GitHub Pages.

## Challenges Faced
I experienced issues with GitHub Pages deployment and accidentally lost the contents of one HTML file while renaming it. I solved these problems by restoring the file, correcting file names, and fixing the GitHub Pages configuration.
